# Skin Disease Detection Project
## Introduction
This project aims to develop a machine learning model that can detect skin diseases from images. The model will be trained on a dataset of images of various skin diseases and will be able to predict the disease with a high degree of accuracy.
## Features
* The model will be able to detect multiple skin diseases, including melanoma, psoriasis, and eczema.
* The model will be trained on a large dataset of images, ensuring that it can generalize well to new, unseen data.
* The model will be able to provide a probability score for each disease, allowing for confident predictions.
## Requirements
* Python 3.8 or later
* TensorFlow 2.4 or later
* Keras 2.4 or later
* OpenCV 4.5 or later
## Usage
1. Clone the repository: `git clone https://github.com/shrvni2/test-project.git`
2. Install the required packages: `pip install -r requirements.txt`
3. Train the model: `python train.py`
4. Evaluate the model: `python evaluate.py`
5. Use the model to make predictions: `python predict.py`
